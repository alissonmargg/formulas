Etiquetas Extras

Etiqueta Pre

La etiqueta pre sirve para respetar lineas y espacios en blanco, al contrario de la etiqueta parrafo.

Hola soy Ali tengo 16 años 

Hola soy
       Ali    tengo     16 años

Etiqueta abr

La etiqueta abr se utiliza para las abreviaturas.

Según datos de la OMS el consumo excesivo de azúcar provoca daños severos en la salud de los niños.

Etiqueta bdo

Si se desea cambiar la orientación de un texto, se utiliza la etiqueta bdo.

Este es un texto normal.

Este es un texto diferente.

Etiquetas cite

La etiqueta cite se utiliza para escrirbir citas textuales.

That"s that me expresso

por: Sabrina Carpenter

Etiquetas small

La etiqueta small hace el texto más pequeño.

Este texto esta dentro de una etiqueta small.

Etiqueta sub

Si desea escribir un subindice, se debe utilizar la etiqueta sub.

1433

Etiqueta sup

Si desea escribir un superindice, se debe utilizar la etiqueta sup.

1433

Etiqueta time

Para escribir fechas y horas se debe utilizar la etiqueta time.

Mañana es día 2024-11-06

Mañana es día 6 de noviembre de 2024

Etiqueta adress

Para especificar al navegador que el texto se refiere a una dirección, se utiliza la etiqueta adress.

Loma dorada No.100 Colonia Lomas de Rosales CP 89100 Tampico, Tamps.
